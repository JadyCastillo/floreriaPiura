# floreriaPiura

Landing page estática para mostrar catálogo de ramos y arreglos florales.

## Tecnologías

- HTML
- Tailwind CSS por CDN
- CSS personalizado
- JavaScript

## Cómo usar

1. Reemplaza las imágenes dentro de `assets/img/`
2. Cambia el número de WhatsApp en `index.html` y `assets/js/script.js`
3. Edita productos en `assets/js/script.js`
4. Sube el proyecto a GitHub
5. Activa GitHub Pages en Settings > Pages

## Publicación en GitHub Pages

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`
